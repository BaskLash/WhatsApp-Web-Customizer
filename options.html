WhatsApp Customizer — Settings

About usage data

This extension sends anonymous usage statistics so we can see which features matter and which to retire. Events are sent to PostHog on EU servers (eu.i.posthog.com) over HTTPS. IP-based geolocation is disabled at the project level, and your IP is not stored against your events.

Collected

  • Which customization features you trigger (event name + categorical metadata)
  • Counts (e.g. number of quick replies)
  • Extension version and your browser language
  • The anonymous ID below

Never collected

  • Messages or anything you type in WhatsApp
  • Contact names, phone numbers, profile pictures
  • Group names or member lists
  • Search queries or chat titles

Your anonymous ID

This random ID is generated locally and is the only identifier attached to your events. Include it in deletion requests so we can remove your data (GDPR / Swiss FADP — right to erasure).
