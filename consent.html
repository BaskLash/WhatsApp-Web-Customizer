Help improve WhatsApp Customizer?

We'd like to collect anonymous usage statistics so we can see which features are useful and which to retire. This is opt-in, fully revocable, and processed in the EU.

What is NEVER collected

  • Your messages or any conversation content
  • Contact names, phone numbers, or profile pictures
  • Group names, group descriptions, or member lists
  • Anything you type inside WhatsApp Web
  • Search queries, chat titles, or media you send/receive

This extension never reads the text on your screen for analytics. The only data we capture is which buttons you click in the extension itself.

What is collected (only with your consent)

  • Which customization features you use (e.g. "theme applied", "quick reply added")
  • Counts and categorical values (e.g. how many quick replies exist, which preset theme was chosen)
  • Extension version and your browser language
  • An anonymous random ID generated on your device — not linked to any account
  • Errors that occur inside the extension

Where it goes

Events are sent to PostHog on EU servers (eu.i.posthog.com) over HTTPS. Nothing else receives this data.

Other network activity (unrelated to analytics)

For full transparency, here's what the extension already does on the network — neither of these is analytics, and both happen regardless of your choice below:

  • Google Fonts. When you select a font from the Typography tab, the corresponding Google Fonts stylesheet is loaded directly from Google's servers. This is required for the font to render.
  • User-pasted image URLs. When you import a background image by URL, the extension fetches that URL directly from your browser. Only the URL you provide is contacted.

You can change your choice any time from the extension's settings page. Read the privacy policy for more.

Either choice closes this tab. Declining is the default — no events are sent until you opt in.
